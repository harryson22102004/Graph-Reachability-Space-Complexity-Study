# Graph Reachability & Space Complexity
Analyzing the memory requirements of fundamental graph algorithms in the context of Complexity Theory.

### 🔬 Research Highlights
- **Savitch’s Theorem**: Simulates the recursive reachability logic used to prove $NSPACE(S(n)) \subseteq DSPACE(S^2(n))$.
- **Log-Space Algorithms**: Implements reachability tests (like BFS/DFS) while monitoring peak auxiliary memory usage.
- **Theory Focus**: Explores the gap between deterministic and non-deterministic space complexity.

**Tech Stack:** Python.
